# Stakeholder Map Template

> Use this template to document stakeholders for each feature.

---

## How to Use

1. Copy this template for each new feature
2. Fill in the stakeholders identified by the Business Analyst
3. Store in `memory/stakeholders.md`
4. Update as new stakeholders are identified
